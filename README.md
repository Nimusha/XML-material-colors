# XML-material-colors
Android XML file, which contains colors from material palette:
https://material.io/design/color/the-color-system.html#tools-for-picking-colors

Structure of color name:
color[shade][b/w]
b - black font
w - white font

realized next colors and shades:
red, 
pink, 
purple,
deep purple,
Indigo, 
Blue, 
Light Blue, 
Cyan, 
Teal, 
Green, 
Light Green, 
Lime, 
Yellow, 
Amber, 
Orange,
Deep Orange, 
Brown, 
Gray, 
Blue Gray, 
Black and White
